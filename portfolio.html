Perivallon (European Project) - Thesis - Illegal Waste Disposal Detection

Period: January 2023 - March 2023

Topics/Skills: CNN, Vision Transformer, Supervised Classification, Multiple Instance Learning (MIL), Remote sensing, Weakly Supervised Segmentation, Illegal Landfill Detection, Tensorflow, Python

Collaboration with: Polytechnic University of Milan, PERIVALLON

Description: I conducted my thesis in collaboration with the PERIVALLON project, co-funded by the European Union. My research focused on detecting illegal waste disposal through satellite image analysis, specifically using supervised classification and weakly supervised segmentation with the AerialWaste dataset from Lombardy, provided by ARPA. The project’s goal was to develop deep learning models to identify illegal landfills across Europe using satellite imagery.
During the project, I faced several technical challenges:
- Scarcity of Annotated Data: I used data augmentation techniques like flipping and noise addition to expand the dataset and enhance model performance.
- Class Imbalance: I addressed this by employing class weights and oversampling to improve landfill detection.
- Varying Image Sizes: To standardize images from different satellites, I created a preprocessing pipeline using bilinear interpolation and zero padding.
- Lack of Localization Annotations: I adopted weakly supervised learning through Hierarchical Heatmap Generation and Multiple Instance Learning (MIL) to localize areas of interest and assign pixel-level probability.
These approaches boosted classification accuracy from 80% to about 90%. Despite limited hardware resources, I developed effective models to handle the extensive data involved.

Applying Vision Transformers for Brain Tumor Classification

Period: September 2024 - October 2024

Topics/Skills: Vision Transformer, Supervised Classification, PyTorch, Python, Self-Attention and Multi-Head Attention mechanisms, Data Augmentation, Early Stopping and Model Checkpointing, Model Training and Evaluation, Visualization Techniques

Description: This project aims to develop a robust machine learning pipeline for classifying brain tumors using MRI images, leveraging advanced techniques such as the Vision Transformer (ViT) model. This work is conducted following the PyTorch for Deep Learning Bootcamp course on Udemy, which provided valuable insights into implementing deep learning models effectively. The project incorporates an enhanced dataset derived from the "Uncovering Knowledge: A Clean Brain Tumor Dataset for Advanced Medical Research" .

Key Features:
● Model Architecture: The project utilizes a Vision Transformer model implemented from scratch, which has demonstrated state-of-the-art performance in various image classification tasks. The model is configured with parameters such as image size, patch size, and multiple transformer layers, providing a robust architecture capable of handling complex patterns in MRI images.
● Training Pipeline:
- Data Loading: Implemented a custom dataset loader using PyTorch's ImageFolder to facilitate efficient loading and transformation of images.
- Data Augmentation: Applied a series of augmentations to enhance the training set's diversity while maintaining sample integrity.
- Training and Evaluation: The training process includes early stopping and model checkpointing based on test accuracy to prevent overfitting and ensure the best model is saved for future inference.
● Visualization:
- Included functions for visualizing training images, class distributions, and model predictions, providing insight into the data and model performance.
- Visualizations help in understanding the distribution of classes in the dataset, which is crucial for ensuring balanced training and evaluation.
● Inference: The trained model is capable of making predictions on new MRI images. The inference pipeline is designed to display images alongside their predicted classes and probabilities, aiding in interpretability.

Challenges and Opportunities in the Use of CLIP - A Comparative Study for Generic, Medical, and Fashion Contexts

Period: August 2023 - October 2023

Topics/Skills: CLIP (Contrastive Language-Image Pre-Training), Image encoder, Caption encoder, Transformer, Python, Language model, Zero-shot classification, Information Retrieval, Image captioning, Tensorflow, Pandas

Collaboration with: Polytechnic University of Milan

Description: I explored the efficacy of Contrastive Language-Image Pretraining (CLIP) models using different text and image encoders. My investigation focused on the COCO dataset, revealing that CLIP performance is significantly influenced by the complexity of these configurations: layers, heads, and embedding size. Additionally, I observed that transformer-based image encoders, especially those with smaller patch sizes, yield better results. Moreover, on the COCO dataset, I utilized the embeddings generated by CLIP encoders to train a decoder, enabling me to perform image captioning tasks with significant results. I leveraged CLIP models with ViT16 and BERT to conduct image retrieval, zero-shot classification, and clustering tasks on the COCO dataset. Subsequently, I extended my study to medical data from the Roco dataset and fashion data from the FashionProduct dataset.

Mathematical Models and Methods for Image Processing

Period: March 2023 - June 2023

Topics/Skills: Python, Denoising, Deblurring, Inpainting, Anomaly Detection, Object Detection, Data-Driven Basis Construction, Sparse Coding and Representation, Robust Fitting Methods, Linear Algebra Applications

Collaboration with: Polytechnic University of Milan

Description: I explored a comprehensive range of topics related to image modeling and processing, emphasizing the use of orthonormal bases and data-driven approaches. The curriculum covered foundational concepts such as Fourier transforms, wavelets, and local polynomial approximation, along with advanced topics like sparsity, redundancy, and the use of redundant sets of generators. Throughout the course, I completed various mini-projects that allowed me to apply theoretical concepts in practical scenarios. These projects included: - Sparse Coding: I implemented algorithms for sparse coding using Orthogonal Matching Pursuit and convex optimization methods like ISTA, IRLS, and LASSO. - Dictionary Learning: I experimented with methods like KSVD to create dictionaries that yield sparse representations for image applications, including denoising, inpainting, and anomaly detection. - Anomaly Detection: I formulated image segmentation problems for detecting anomalies and utilized nonparametric methods along with data-driven bases for effective results. - Robust Fitting Methods: I learned about and implemented robust fitting techniques, including RANSAC, LMEDS, and the HOUGH transform, for object detection in images. The course also provided insights into linear algebra concepts relevant to image models, inverse problems, and the application of sparse representations in various image processing tasks. Overall, this hands-on experience deepened my understanding of the mathematical foundations and practical applications of image modeling techniques.

Question Answering Project - Natural Language Processing (NLP)

Period: April 2023 - May 2023

Topics/Skills: Language model, Transformer, Bert, Gpt2, Question Answering, Pytorch, T5, Pandas, Python

Collaboration with: Polytechnic University of Milan

Description: I engaged in an exciting Natural Language Processing project that centered around the Question Answering task, utilizing the renowned SQuAD (Stanford Question Answering Dataset). This project presented a unique opportunity to explore various aspects of question answering. One of the key tasks was to determine whether a given answer could be found in the text in response to a specific question. This involved analyzing the relationship between the question and the provided text to assess the presence of the answer. Additionally, I focused on extracting the answer directly from the text using the BERT model, which is well-suited for this type of task due to its ability to understand context and semantics. Another important aspect of the project involved generating responses, which could either be contextually informed by the text or independent of it, by utilizing generative models. To achieve these objectives, I employed several state-of-the-art NLP models, including BERT, T5, and GPT-2. Rather than implementing these models from scratch, I fine-tuned them on the SQuAD dataset using the Hugging Face library, which allowed me to leverage pre-trained architectures and adapt them effectively for the specific requirements of the project.

Time Series Forecasting

Period: Dicember 2021 - Genuary 2022

Topics/Skills: Time Series Forecasting, LSTM (Long Short-Term Memory), RNN (Recurrent Neural Network), Data Preprocessing, Model Training, Performance Evaluation, Sequence Modeling, Python, TensorFlow

Collaboration with: Polytechnic University of Milan

Description: I focused on predicting seven features from time series data with 65,528 time steps, aiming to forecast the next 864 steps. To achieve this, I developed several types of neural networks, including Residual Bidirectional LSTM and Stacked Bidirectional LSTM.
Key Steps:
- Data Preparation: I segmented the long time series into smaller sequences for better learning.
- Pre-processing: I applied techniques to scale the data and ensure it was stationary.
- Model Training: I trained the models to improve their prediction accuracy by adjusting data splits and settings.
The best-performing model, the Stacked Bidirectional LSTM, achieved impressive results with a Root Mean Square Error (RMSE) of 4.26 and a Mean Absolute Error (MAE) of 2.88. This project demonstrated the effectiveness of deep learning techniques for time series forecasting.

Leaves Classification

Period: Dicember 2021 - Genuary 2022

Topics/Skills: Image Classification, Convolutional Neural Networks (CNN), Data Augmentation. Model Training, Hyperparameter Tuning, Performance Evaluation, Python, TensorFlow

Collaboration with: Polytechnic University of Milan

Description: I worked on classifying images of different types of leaves. The dataset included 17,728 images of size 256x256 pixels, divided into 14 classes such as Apple, Grape, and Tomato. The project involved two main steps: feature extraction using a Convolutional Neural Network (CNN), specifically the Inception V3 model, and classification of the leaves.
Key Steps:
- Dataset Preparation: Due to class imbalance, I enhanced the dataset to include 79,702 images by applying Random Oversampling and data augmentation techniques like rotation and brightness adjustments to create variations of the original images.
- Data Augmentation: I introduced transformations to help the model generalize better and reduce overfitting during training.
- Model Training: The Inception V3 model was fine-tuned for classification. I used a custom output layer suitable for the 14 classes, applying techniques like dropout to mitigate overfitting.
- Evaluation: The model was trained to minimize the Categorical Cross-entropy loss function, and its performance was measured using accuracy and the F1 score. The best model achieved an accuracy of 83.21% on the test set.
This project demonstrates the application of deep learning techniques for effective image classification, highlighting the importance of data preparation and model tuning in achieving high accuracy.

Prediction of Solar Radiation Through Machine Learning Algorithms

Period: September 2017 - October 2018

Topics/Skills: Feature Selection, Random Forest, Support Vector Machine, Multi-Layer Perceptron, AutoRegressive Moving Average, Long-Short Term Memory, Meteorological Data, Precision Agriculture, Solar Radiation Prediction

Collaboration with: Polytechnic University of Milan

Description: This project explores the use of advanced technology and machine learning to enhance decision-making in agriculture. By collecting extensive meteorological data through the Internet of Things (IoT), farmers can optimize crop management, irrigation, and resource allocation. Focusing on solar radiation prediction across multiple time horizons (hourly, daily, monthly), we employ machine learning algorithms such as Support Vector Machines, Multi-Layer Perceptron Neural Networks, and Long Short-Term Memory (LSTM) networks for accurate forecasts. Feature selection techniques, including Pearson correlation, Recursive Feature Elimination, and Random Forest, are applied to identify the most relevant meteorological parameters. The goal is to determine the best combination of algorithms and parameters for precise solar radiation predictions, ultimately supporting effective irrigation strategies and enhancing agricultural productivity.
